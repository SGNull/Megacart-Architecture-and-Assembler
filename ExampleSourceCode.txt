// Fibbonacci sequence generator program.
// It stops before it sees an overflow.

// Starts by setting A to 1, B to 0, and giving the IO these first two numbers.
ALU A ONE
MFA B
ALU B DECB
MFB IO
ALU B INC

// Here is the main loop. It goes back and forth, adding numbers. The ADDS operation is addition & store carry.
: LOOP
MFA IO
ALU B ADDS
JIF CAR END

MFB IO
ALU A ADDS
JIF NCAR LOOP

// The end of the program
: END
